# Train Routing Frontend

Vue.js 3 frontend for the MOB Train Routing application.

## Tech Stack

- Vue.js 3
- Vuetify 3
- TypeScript 5
- Vite
- Pinia (state management)
- Vue Router

## Development

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:3000)
npm run dev

# Build for production
npm run build

# Run tests
npm run test

# Lint code
npm run lint
```

## Project Structure

```
src/
├── components/     # Reusable Vue components
├── layouts/        # Page layouts
├── pages/          # Route pages (file-based routing)
├── plugins/        # Vue plugins (Vuetify, Pinia, etc.)
└── styles/         # Global styles
```
